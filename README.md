초코비 먹고 싶은 짱구는 20초 내에 20개의 초코비를 수집하는 클릭형 게임입니다. 사용자는 마우스를 움직여 짱구 캐릭터를 조작하며, 화면에 나타나는 초코비를 클릭해 점수를 획득합니다. 모든 초코비를 수집하거나 시간이 종료되면 게임이 끝나며, 결과 메시지가 표시됩니다.

🕹️ 진행 방식
게임을 시작하면 짱구가 마우스를 따라다닙니다

화면에 초코비가 20개 나타납니다

제한 시간 (20초) 안에 초코비를 모두 클릭하면 게임 클리어!

시간 내에 다 못 클릭하면 "Time Over" 메시지가 뜹니다

점수와 결과 메시지가 화면에 표시됩니다
